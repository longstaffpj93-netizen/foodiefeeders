# 🍔 FoodieFeeders Hartlepool (TS24)

Professional, homemade food delivery service based in Hartlepool. This repository contains the source code for the FoodieFeeders web application, featuring a modern, dark-themed UI and seamless WhatsApp ordering integration.

## 🚀 Live Demo
The site is hosted and can be viewed at:
- **Primary:** [foodiefeeders.tiiny.site](https://foodiefeeders.tiiny.site/)
- **GitHub Pages (Mirror):** `https://longstaffpj93-netizen.github.io/foodiefeeders/` (Pending Setup)

## ✨ Features
- **Responsive Design:** Fully optimized for mobile (Android/iOS) and desktop.
- **Real-time Status:** Automatically detects if the kitchen is open (Sat/Sun or 5PM-10PM Mon-Fri).
- **Interactive Cart:** Real-time subtotal calculation and delivery fee inclusion.
- **WhatsApp Integration:** One-click order submission with pre-formatted delivery details.
- **Modern UI:** Built with Tailwind CSS, featuring glassmorphism and smooth animations.

## 🛠️ Tech Stack
- **Frontend:** HTML5, Tailwind CSS, JavaScript (Vanilla)
- **Deployment:** Git/GitHub, Tiiny Host
- **Communication:** WhatsApp Business API

## 📂 Project Structure
- `index.html` - Main application logic and UI.
- `*.jpeg` - High-quality food assets and menu photography.
- `foodiefeeders.zip` - Deployment package for Tiiny Host.
